=====================
MEMO vite development
=====================

Local install::

  # in folder ./client/modern/
  $ npm install

Start development server::

  $ ./manage vite.modern.dev

  # in folder ./client/modern/
  $ npm exec -- vite

Fix source code::

  # in folder ./client/modern/
  $ npm run fix

Fix & Build::

  $ ./manage vite.modern.build
